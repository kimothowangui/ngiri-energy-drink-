## ğŸš¨ Troubleshooting

**Port 3000 already in use?**
- Vite will auto-assign port 3001, 3002, etc.
- Check the terminal output for the actual port

**Images not loading?**
- Ensure `img/` folder exists with web1.PNG through web7.PNG
- Check browser console (F12) for errors

**Animations laggy?**
- Close other browser tabs
- Disable browser extensions temporarily
- Use hardware acceleration if available

**Nothing happens when dragging on Shop page?**
- Make sure JavaScript is enabled
- Try clicking and holding, then dragging slowly
- Works on touch devices too!
